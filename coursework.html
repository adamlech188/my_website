Introduction to Mathemathical Proofs
Although this class didn't teach me directly programming and software development, nevertheless it was very important one to further my understading of Computer Science and programming. This class exposed me for the first time to mathematical thinking using strict formal logic.It taught me to operate on more abstract and conceptual level. I was introduced to set theory , number theory, and function with emphasis on logical justifying claims about mathematical objects. As it turned out, it had enormous importance in Computer Science. This course made me aware how much we rely on unproven assumptions and how to be critical about them.
Introduction to GUI Programming and Graphics
Design and implementation of object-oriented graphical user interface and computer graphics system. Projects involving basic principles of concurrent and event-driven programming.
The project in this class, provided full game graphics and user interface. I created all geometrical figures and shapes as components of the game starting first with primitive geometrical shapes in 3D developed applying fundamental algorithms for clipping, scan conversion, viewing affine and convex linear transformations, projections and viewing. After that, the components of the user interface were created: buttons, text fields and text areas. This project gave very thorough understanding of how the components of graphical user interface are developed. Since this project simulated a real time game with player connected on the remote server, I also had to apply basic principles of concurrent programming and thread management that exposed me to programming problems in broader computer-system context.
